# 墨流し（仮称）

> 眠れても眠れなくても、心が乱れていない——そのための、就寝前の所作アプリ。
> Ataraxia Works の一員。**測らない・貯めない・煽らない。**

巷の睡眠アプリは「測る」。だが測ること自体が自己採点を生み、眠りを乱す。墨流しは逆を行く。中心に置くのは「よく眠ること」ではなく「心が乱れていないこと」。よく眠れるのは結果としてのオマケ。

夜、心の引っかかりをひとことだけ書く → 水面に浮かぶ → 下へ払う（または待つと自分から溶ける）→ 墨のように水中へ散って消える → 静まる。それだけ。

## ataraxia works との統一

- **単一ファイルの素の PWA**（フレームワークなし・外部依存なし・ビルドなし）。`index.html` / `manifest.json` / `sw.js` / `icon.svg`。
- 共通デザイントークン（色変数名・フォント・角丸・`--shadow`・`env(safe-area-inset-*)`・`rise`アニメ・`prefers-reduced-motion`尊重）を継承。
- アイコンは家族の文法（`rx=14` の角丸ダーク四角＋白のグリフ）。墨流しは「一滴＋水中へ拡散」。
- ナビ・設定シート・声（責めない／端末の中だけ）も家族に合わせる。
- **唯一の意図的な逸脱：パレットをダークに反転**。就寝前に白画面は目に痛く、墨×水はダークで最も映えるため。形・余白・文法はそのまま＝一目で家族の一員。

## 思想の実装メモ

- **本文は一切保存しない**。書いた引っかかりは「書く→流す」の間だけメモリ上に存在し、散った瞬間に消える。端末に残すのは `lastFlowDate`（ゆうべ流した日付）と `morningPending` だけ。設計書 v0.1 の「一晩ローカル保持」より一段強い。
- 連続記録・スコア・履歴・バッジ・通知・共有は**作らない**（やらないことリスト）。
- 朝の確認は「のこってる／消えた」の2択（＝採点）をあえて外し、本文も出さず、`ゆうべ、ひとつ流しました。それだけで、じゅうぶん。` の静かな一言だけにした（オープンクエスチョン3への暫定回答）。

## まだ決まっていないこと（設計書のオープンクエスチョン）

1. 正式名称（「墨流し」/ サブドメイン。当面 `sumi` を作業名に）
2. 手放す操作 — 本実装は「払う＋9秒で自動溶解」の両対応
3. 朝の確認の扱い — 上記の暫定版。完全に無くす案も残す
4. 就寝画面の中身 — いまは暗転＋ごく淡い呼吸の点。完全な無にする案も
5. マネタイズ — 買い切り想定（サーバーレス＝限界コスト0で成立）
6. 対話風レイヤー — コアから分離して post-launch

## デプロイ（家族と同じ：Cloudflare Pages + TWA）

1. このリポジトリを Cloudflare Pages に接続（フレームワークなし／ビルドコマンド空欄／出力 `/`）
2. カスタムドメイン（例：`sumi.xdcyw.net`）を割り当て
3. TWA 署名は共有 `ataraxia.jks`（alias `ataraxia`、パッケージ `net.xdcyw.sumi`）。`icons/icon-{192,512}.png` と `.well-known/assetlinks.json` は用意済み。Android 版は `sumi.apk` としてこのリポジトリに同梱（landing から DL）。
